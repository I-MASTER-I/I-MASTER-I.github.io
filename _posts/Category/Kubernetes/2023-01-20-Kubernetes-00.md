# 쿠버네티스 환경 구성

## 1. 가상환경 구성
- 가상환경 도구
    - vmware fusion 13
- OS : ubuntu 20.04 LTS Desktop
- 환경 구성
    - master
    - node1
    - node2

## 2. 가상환경 세팅
1. apt-get 명령어 update
```
sudo apt-get update
```

2. ssh 설치
```
sudo apt-get install openssh-server
```

3. ip정보를 위한 net-tools 설치
```
sudo apt install net-tools
```

## 3. 접속테스트
- ssh 접속
```
ssh <계정명>@<server ip>
```
