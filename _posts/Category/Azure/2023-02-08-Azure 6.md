# Azure Static Web Apps 개요
- 풀 스택 웹 앱 자동 빌드 + Azure 배포 서비스
    - SWA의 워크플로우 조정
    - Repo의 선택한 브랜치 모니터링
    - 선택 브랜치에 커밋 푸시 또는 PR 수락 > 자동빌드 > Azure에 앱과 API 배포
- 라이브러리와 프레임워크
    - 서버 쪽 렌더링이 필요하지 않은 라이브러리 및 웹 프레임워크 사용
    - HTML, CSS, JavaScript, 이미지
